⚡ Prédiction de puissance à Dakar Déployé Prototype

Application web de prédiction des risques de coupures d'électricité dans 8 quartiers de Dakar. Utilise deux modèles de Machine Learning : LightGBM (gradient boosting, ~88% précision) et LSTM (deep learning, ~90% précision) entraînés sur 70,001 enregistrements de données synthétiques.

Technologies:
Python LightGBM TensorFlow LSTM Streamlit Plotly

📸 Aperçu

💡 Applications potentielles

Ce prototype pourrait être adapté pour :

  • Planifier les interventions de maintenance préventive
  • Anticiper les zones à risque avant les pics de consommation
  • Améliorer la satisfaction client grâce à une meilleure disponibilité
  • Optimiser l'allocation des équipes techniques

📌 Note : Prototype développé avec des données synthétiques. Évolution possible avec données opérationnelles réelles.
